# EarthMod
允许用户将OSM地图的瓦片贴图信息加载在ParacraftWorld中草地层上方(第五层)  
可选学校信息依赖于运营端添加的学校数据  
同时由于需要用到内嵌浏览器，所以需要同时加载另外两个Mod,具体操作是手动将extra文件夹中的两个zip包  
复制到paracraft根目录的Mod文件夹下  
  
部署操作文档及使用说明书参照下述链接  
[参考文档](https://github.com/idreamtech/EarthMod/tree/master/Doc)
